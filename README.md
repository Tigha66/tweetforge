# TweetForge - AI Tweet Generator

Turn your ideas into viral tweets instantly.

## Features
- AI-powered tweet generation
- Thread maker
- Bio writer
- Multiple tones (professional, casual, funny, viral)

## Pricing
- Free: 5 tweets/day
- Pro ($3): Unlimited tweets + thread export

## Deploy
```bash
vercel --prod
```

## Environment
- PERPLEXITY_API_KEY (optional, uses fallback prompts)
